## boxlite ssh

SSH into a box

### Synopsis

Establish an SSH connection to a running box

```
boxlite ssh [BOX_ID] | [BOX_NAME] [flags]
```

### Options

```
      --expires int   SSH access token expiration time in minutes (defaults to 24 hours) (default 1440)
```
