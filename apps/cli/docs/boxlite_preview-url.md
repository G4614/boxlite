## boxlite preview-url

Get signed preview URL for a box port

```
boxlite preview-url [BOX_ID | BOX_NAME] [flags]
```

### Options

```
      --expires int32   URL expiration time in seconds (default 3600)
  -p, --port int32      Port number to get preview URL for (required)
```
